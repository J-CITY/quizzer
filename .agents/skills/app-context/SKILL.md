# Контекст проекта Quizzer (Agent Context)

## Описание
Quizzer — это кроссплатформенное мобильное приложение на Flutter для изучения японского и любых других языков при помощи продвинутой системы карточек. 
Приложение позволяет учить слова из Google Таблиц, переводить и добавлять новые слова на лету, собирать ответы из букв или вводить их текстом, а также отслеживать свою активность и стрик (непрерывные дни занятий).

## Стек технологий
- **Фреймворк**: Flutter (Material 3)
- **Стейт-менеджмент / DI**: `flutter_riverpod`
- **Локальная база данных**: `isar`
- **Фоновые задачи (уведомления)**: `workmanager`, `flutter_local_notifications`
- **Аудио и TTS**: `flutter_tts` (озвучка на разных языках), `audioplayers` (звуковые эффекты)
- **Локализация**: Стандартный `flutter_localizations` (`.arb` файлы, *ВАЖНО: новые текстовые строки всегда нужно добавлять сразу в файлы локалификации*).
- **Графика и анимации**: `fl_chart` (статистика), `lottie` (анимации стриков). Кастомный инструмент `LottieColorShift` для парсинга и динамической перекраски Lottie анимаций.
- **Интеграции и API**: `translator` (Google Translate), Jisho.org API (чтение кандзи), Pixabay API (для получения картинок к словам).

## Структура проекта
- `lib/data/models/`: Модели Isar (`Word`, `CustomList`, `Settings`, `TrainingSession`). 
- `lib/data/services/`: Работа с БД (`DatabaseService`), сетью и парсингом Google Sheets (`GoogleSheetsService`), уведомлениями (`NotificationService`), профилем (`ProfileService`).
- `lib/domain/`: `TrainingEngine` — бизнес-логика формирования сессии вопросов, выборки слов (Левенштейн, испорченные слова, конструктор).
- `lib/ui/`: Экраны (поиск, списки, настройки, статистика, тренировка) и переиспользуемые виджеты (`GlowButton`, `AcrylicCard`, `LanguageSelector`).
- `lib/utils/`: Вспомогательные утилиты, включая утилиты для проверки языков TTS (`LanguageUtils`) и генерации испорченных вариантов ответов.

## Основные фичи и логика

1. **Мультиязычность и Поиск (`SearchScreen`)**:
   - Пользователь может переводить слова через Google Переводчик прямо в приложении.
   - Для японского языка (ja) автоматически подтягивается чтение (хирагана/катакана) через API `jisho.org`.
   - Можно добавлять свои кастомные языки для изучения, они валидируются на поддержку TTS (`LanguageUtils`).

2. **Интеграция с Google Sheets**: 
   - Загрузка слов (Слово, Чтение, Перевод) напрямую из публичных CSV/Google таблиц.
   - Пользовательские списки (`CustomList`) могут синхронизироваться с таблицами и иметь свои уникальные настройки тренировки (переопределять глобальные).

3. **Продвинутая логика тренировки (`TrainingEngine`)**: 
   - **Разные типы вопросов**: Классические карточки (4 варианта ответа), восприятие на слух (Voice-to-Word), угадывание по картинке (Image-to-Word, работает через Pixabay).
   - **Конструктор и Ввод (Input)**: Пользователь может вводить ответ текстом или собирать его по буквам (Конструктор, куда подмешиваются буквы из других слов).
   - **Умные неправильные ответы**: Используется алгоритм Левенштейна (`useSimilarWordsForOptions`) для подбора похожих слов и система "порчи" правильного слова (`useSpoiledWordsForOptions`) на основе таблиц похожих символов.
   - Прогрессия (0..5 баллов). Ошибки сбрасывают прогресс в 0.

4. **Статистика и Стрики**:
   - Графики `fl_chart` (BarChart для дней, PieChart для списков).
   - Анимированный Lottie-огонек стрика перекрашивается на лету (HSL) в зависимости от крутости (количества) дней подряд.

5. **Уведомления**:
   - Регулярные "пинги" через `workmanager` с учетом разрешенного интервала ("с 10:00 до 22:00").

## Важные команды для работы
- Перегенерация Isar моделей: `dart run build_runner build -d`
- Перегенерация локализации: `flutter gen-l10n`
