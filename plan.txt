Хочу написать приложение для тренировки японских слов.

Слова должны браться из гугл таблицы.
Её формат: слово/словосочетание, слово/словосочетание на хирагане/катакане, перевод на русский.
Второго столбца может не быть.

При открытии приложения актуализируются данные из гугл таблицы.
Если нет сети, то используется закэшированная версия слов. Всплывает toast, что не получилось обновить слова.

В верху экрана вкладки приложения. Пока их 2: все слова и кастомные группы слов.

Экран 1: Все слова.
В верху поиск. При наборе фильтрует слова. Поиск осуществляется как на русском, так и на японском
Далее список всех слов.
Элемент списка:
Первая строка: слово на японском - круглый прогресс бар - чек бокс
Вторая строка (более мелкий шрифт): перевод слова
Если у слова прогресс 100 процентов или стоит галочка, то слово подсвечивается зелёным.
Внизу, справа кнопка плей - запускает тренировку.

Круглый прогресс бар - отображает прогресс выучивания слова.
Чек бокс - можно отметить, что знаешь слово, прогресс выучивания становится 100 процентов. При снятии чекбокса прогресс падает в 0. По умолчанию чекбокс не стоит и прогресс 0.
При нажатии на элемент - открывается карточка, со всей инфорацией о слове.

Экран 2: Кастомные списки слов
Внизу справа - кнопка '+' - добавить новый список.
При нажатии открывается окошко, где ввести название списка и ниже список всех слов.
Слева чекбокс, при выборе чекбокса, слово добавляется в список, при снятии - убирается.

На жкране выводятся все кастомные списки слов.
Элемент списка: Название списка - кнопка настроек

При нажатии на список, он открывается, там список его слов. Список отображается так же как и на 1ом окне.
Внизу справа - тоже кнопка плей, тренировать этот список.

При нажатии кнопки настроек, открывается окно, как при добавлении списка, но не для нового списка, а для выбранного.
Там можно изменить название и отредактировать список слов для списка.

Процесс тренировки:
Каждая тренировка - это 50 вопросов.
При нажатии на кнопку плей запускается тренировка.
Из списка (если запущен кастомный саисок) или из всех слов выбирается 50 слов (слова могут повторяться, если в списке меньше 50 слов)
Формируются карточки вопросов.
Карточка вопросов - слово, ниже слова - 4 варианта ответа. При нажатии на ответ, через секунду, запускается следующий вопрос. Если ответ правильный выбранный ответ подсвечивается зелёным, иначе красным. 
Если ответил неправильно, то вопрос помещяется в конец списка и покажется ещё раз.
Справа вверху от слова - маленькая круглая кнопка, проиграть слово голосом. Слово проигрывается именно на японском, даже если вопрос на русском. Голос делается через какой-нибудь синтез речи.
В конце показывается окно со словами, где были ошибки и кнопка закрыть. После закрытия попадаешь на главный экран.

Варианты вопросов:
слово - слово на катакане/хирагане
слово на катакане/хирагане - слово
слово + слово на катакане/хирагане - перевод
перевод - слово + слово на катакане/хирагане

Вариант вопроса выбирается случайно.
Неверные 3 варианта выбираются случайно.

Процесс за запоминания слов:
Если в тренировке не была допущена ошибка для слова, то к прогдессу добавляется 1. 
Если была, то прогресс сбрасывается в 0.
Если прогресс == 5, то слово выучено, и оно больше не учавствует в тренеровке.
Если прошла неделя с последней тренеровки слова с полным прогрессом, то у него отнимается 1 еденица прогресса. (Пора слово повторить)

Добавь нотификации:
В дневное время каждый час присылай нотификацию что пора повторить слова

Добавь окно настроек:
Там же, где рисуются вкладки справа добавь кнопку 'три точки'. По нажатию открывается меню, там настройки.
В настройках пока только 3 параметра.
Поменять количестко слов в тренировке, по умолчанию 50
Установить id гугл таблицы, откуда качать слова
Включить/отключить нотификации и выбрать период отправки нотификации, по умолчанию 1 час

Основные окна:
Теперь на главном экране список списков слов.
Из настроек убрать id google таблицы
При добавлении нового списка, там можно опционально указать id таблицы. И тогда в новый список выкачивается эта таблица слов.
Если не получилось загрузить, то показывать toast с ошибкой.
Так же кнопка  переключатель, нужно ли грузить при старте приложения.
При включении приложения не нужно каждый раз обновлять все таблицы, а только те, у которых стоит параметр "грузить при старте".
При создании нового списка, там больше нет списка всех слов. Он создается пустой или (если есть google таблица), то со словами из таблицы.

При открытии списка, если это список с google таблицей, то вверху есть кнопка, для обновления таблицы.
В настройках списка (кнопка вверху с тремя точками):
  Кнопка "Удалить", чтобы удалить список.
  Если это список без google таблицы, то кнока "Редактировать", чтобы можно было удалить слова из списка
  Кнопка "Добавить в другой список", при нажатии можно выбрать слова из списка и появляется кнопка "Добавить".
  При нажатии "Добавить", открывается список списков списков слов без google таблицы, при нажатии на элемент списка, в выбранный список добавляются выбранные слова.

При долгом нажатии на элемент списка, оно копируется в буфер обмена. Показывается toast, что скопировано.

То есть, удалять и добавлять слова можно только в списки без google таблицы.
При добавлении слов из списка с google таблицей, слова копируются в этот список. То есть, если слово удалится их google таблицы, то в "локальных" списках оно останется.
